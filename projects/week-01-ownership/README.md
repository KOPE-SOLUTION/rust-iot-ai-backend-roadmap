# Week 1 — Ownership and Borrowing

## Objective
Understand Rust ownership model and borrowing rules.

## Topics
- Ownership move
- Clone vs move
- Immutable reference (&)
- Mutable reference (&mut)
- Borrowing rules

## Key Rules

- Ownership moves by default
- Use `clone()` to copy data
- Use `&` to borrow
- Use `&mut` to modify data
- Cannot have mutable and immutable references at the same time

## How to Run

```bash
cargo run
```
