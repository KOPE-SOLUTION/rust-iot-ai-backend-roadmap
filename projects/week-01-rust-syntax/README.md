# Week 1 — Rust Syntax Basics

## Objective
Learn basic Rust syntax including:
- Variables
- Functions
- Conditionals
- Loops
- Ownership

## How to Run

```bash
cargo run
```

## Key Learning

- Rust variables are immutable by default
- Ownership prevents memory bugs
- Functions require explicit types
- Loops and conditions are similar to other languages but safer
