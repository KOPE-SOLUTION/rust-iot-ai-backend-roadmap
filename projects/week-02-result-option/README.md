# Week 2 — Practice Result and Option patterns

## Objective

Practice Rust `Option` and `Result` patterns using IoT-style sensor examples.

## Concepts Covered

- `Option<T>`
- `Result<T, E>`
- `match`
- `unwrap_or`
- `ok_or_else`
- `?` operator

## Example Use Cases

- Find a sensor unit from a supported list
- Parse numeric sensor values safely
- Build a validated sensor reading
- Handle unsupported sensors or invalid input

## How to Run

```bash
cargo run
```

## How to test

```bash
cargo test
```
