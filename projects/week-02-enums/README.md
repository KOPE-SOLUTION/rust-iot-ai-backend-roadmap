# Week 2 — Enums for Status and Severity

## Objective

Replace string-based status and severity with Rust enums for type safety.

## Enums

- `Severity`
  - Low
  - Warning
  - Critical

- `EventStatus`
  - Active
  - Resolved

## Benefits

- Prevent invalid values
- Compile-time safety
- Cleaner logic with match
- Easier to maintain

## How to Run

```bash
cargo run
```

## How to Test

```bash
cargo test
```
