# Week 4 — Axum API Basics

## Objective

Build a Rust backend server using Axum with basic IoT API endpoints.

## Features

- `GET /health`
- `GET /devices`
- `GET /telemetry/latest`
- tracing logs
- SQLite database integration

## Tech Stack

- Rust
- Axum
- Tokio
- SQLx
- SQLite
- tracing

## Run

```bash
cargo run
```

## Endpoints

### Health

```bash
GET /health
```

### Devices

```bash
GET /devices
```

### Latest Telemetry

```bash
GET /telemetry/latest
```
